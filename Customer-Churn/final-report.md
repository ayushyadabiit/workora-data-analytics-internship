# Final Report

## Project Summary

This project analyzed customer churn data to identify the major factors affecting customer retention.

## Work Completed

- Dataset exploration
- Data validation
- Exploratory data analysis
- Business insights generation
- Churn prediction planning

## Outcome

The analysis highlighted customer tenure, contract type, and monthly charges as important factors influencing churn.

## Conclusion

Understanding customer churn enables organizations to improve customer satisfaction, reduce attrition, and increase long-term profitability.
