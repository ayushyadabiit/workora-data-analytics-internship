# Exploratory Data Analysis

## Analysis Performed

- Checked dataset dimensions
- Verified missing values
- Examined data types
- Calculated churn distribution
- Compared monthly charges with churn
- Analyzed contract types
- Studied customer tenure

## Key Findings

- Customers with month-to-month contracts showed higher churn.
- Higher monthly charges were associated with increased churn.
- Long-term customers were more likely to stay.
- Electronic check users had relatively higher churn.
